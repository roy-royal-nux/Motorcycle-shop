# Iron & Chrome - Custom Motorcycle Shop Website

A modern, responsive website for a traditional motorcycle custom build shop.

## Features

### 🎨 Design
- **Professional Color Scheme**: Red, black, and orange colors representing a traditional motorcycle shop
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, modern interface with smooth animations and transitions

5. **About Section**

6. **Contact Section**

7. **Footer**

### 🎯 Customization

To customize for your motorcycle shop:

1. **Colors**: Update CSS variables in `:root`
   ```css
   --primary-color: #c41e3a;      /* Red accent */
   --secondary-color: #1a1a1a;    /* Black */
   --accent-color: #f7931e;       /* Orange */
   ```

2. **Content**: Update text in HTML sections
   - Shop name and logo
   - Motorcycle build names and descriptions
   - About section content
   - Contact information
   - Social media links

3. **Branding**: Replace placeholder bike images
   - Update bike-image divs with real images
   - Modify gallery gradients to match your style

4. **Contact Form**: Integrate with a backend service
   - Currently shows success/error messages
   - Connect to email service (Formspree, EmailJS, etc.)

## Getting Started

3. All sections are interactive - click navigation to explore
4. Mobile menu: Click hamburger icon on smaller screens
